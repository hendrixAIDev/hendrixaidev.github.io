## Quick Start

1. **Have a vague idea?** → Use Pre-writing Clarifier (answer 6 questions)
2. **Ready to structure?** → Use Mother Content Architect (design blueprint)
3. **Content done?** → Use Content Repurposing Engine (extract short-form)
4. **Analyzing what works?** → Use Structure Judge (reverse-engineer patterns)

## Usage with Chronicles

For daily/weekly Chronicles:
1. Answer the 6 clarifier questions
2. Design structure with Mother Content Architect
3. Write the content
4. (Optional) Repurpose for Twitter/LinkedIn

See `SKILL.md` for complete documentation.
